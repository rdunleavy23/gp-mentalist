# Logo Upload Instructions

## Required Logo Files

Add these 5 logo files to this directory (`/public/images/logos/`):

1. **luke-bryan.png** - Luke Bryan logo (white text on black)
2. **texas-am.png** - Texas A&M University logo (white on dark green)
3. **chick-fil-a.png** - Chick-fil-A logo (white script on black)
4. **southwest-airlines.png** - Southwest Airlines logo (white text with heart icon on dark green)
5. **us-air-force.svg** - ✅ Already added (SVG format)

## File Requirements

- **Format**: PNG for most logos, SVG for US Air Force (already added)
- **Size**: 200px width minimum (2x for retina = 400px)
- **Background**: Transparent PNG preferred, or match brand colors
- **File Size**: Optimize to <50KB each (use TinyPNG or ImageOptim)
- **Naming**: Exact filenames as listed above (lowercase with hyphens)

## How to Add Logos

1. Save/export logos as PNG files (except US Air Force which is SVG)
2. Name them exactly as listed above
3. Place them in `/Users/ryan/gp-mentalist/public/images/logos/`
4. The component will automatically display them

# Quick Logo Search Reference

## One-Liner Prompt for LLM

```
Find official SVG logos for: Southwest Airlines, Alcon, Chick-fil-A, Ferrari, Luke Bryan, Triumph, Olipop. Provide direct download links or complete SVG code. Verify sources are official. Format: transparent background, full color, high quality.
```

1. **Simple Icons** (fastest option):
   - https://simpleicons.org/icons/southwestairlines.svg
   - https://simpleicons.org/icons/ferrari.svg
   - https://simpleicons.org/icons/chickfila.svg
   - https://simpleicons.org/icons/alcon.svg
   - https://simpleicons.org/icons/triumph.svg
   - (Search for others: https://simpleicons.org/)

2. **GitHub Simple Icons Repository**:
   - https://github.com/simple-icons/simple-icons/tree/develop/icons
   - Direct SVG files in `/icons/` folder

3. **Company Official Pages**:
   - Southwest: https://www.swamedia.com/
   - Ferrari: https://corporate.ferrari.com/en/media
   - Triumph: https://www.triumphmotorcycles.com/

## Quick Terminal Commands

```bash
# Download from Simple Icons (if available)
curl -o southwest-airlines.svg https://simpleicons.org/icons/southwestairlines.svg
curl -o ferrari.svg https://simpleicons.org/icons/ferrari.svg

# Or use GitHub raw URLs
curl -o southwest-airlines.svg https://raw.githubusercontent.com/simple-icons/simple-icons/develop/icons/southwestairlines.svg
```

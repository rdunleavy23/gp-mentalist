# Grant Price Photos

Add the following photo files to this directory:

1. **grant-stage-performance.jpg** (or .png)
   - Stage performance photo with Grant and audience member
   - Use for: Hero section or Video section thumbnail
   - Recommended: 1600px × 1200px (4:3 aspect ratio)
   - Optimize for web (target: <300KB)

2. **grant-headshot.jpg** (or .png)
   - Professional headshot/profile photo of Grant
   - Use for: About section
   - Recommended: 800px × 1000px (3:4 aspect ratio for portrait)
   - Optimize for web (target: <200KB)

## Photo Requirements

- **Format**: JPG (preferred for photos) or PNG
- **Optimization**: Use tools like TinyPNG, ImageOptim, or Squoosh
- **Quality**: High quality but web-optimized
- **Aspect Ratios**: Maintain as specified above
- **Color Space**: sRGB for web display

## Usage

The components are already configured to use these images. Once you add the files with the exact names above, they will automatically appear on the site.
